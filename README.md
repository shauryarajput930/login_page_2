# Login Page 2

A modern, responsive authentication page with smooth animations and social login options.

## Features

- **Dual Authentication Forms**: Switch between Sign In and Sign Up seamlessly
- **Social Media Integration**: Quick login options for Facebook, Google, and LinkedIn
- **Responsive Design**: Optimized for both desktop and mobile devices
- **Smooth Animations**: Elegant sliding panel transitions
- **Modern UI**: Clean, professional interface with gradient backgrounds

## Technologies Used

- **HTML5**: Semantic markup and form structure
- **CSS3**: Modern styling with animations and transitions
- **JavaScript**: Interactive form switching and user experience enhancements
- **Font Awesome**: Social media icons

## File Structure

```
login_page_2/
├── index.html          # Main HTML structure
├── style.css           # Styling and animations
├── style.js            # JavaScript functionality
└── README.md           # Project documentation
```

## How to Use

1. Clone or download the project files
2. Open `index.html` in your web browser
3. Navigate between Sign In and Sign Up forms using:
   - The sliding panel buttons (desktop)
   - The mobile switch buttons (mobile)
4. Fill in the required fields to register or login

## Features in Detail

### Authentication Forms
- **Sign Up Form**: Full name, email, and password fields
- **Sign In Form**: Email and password fields with "Forgot password" link
- **Form Validation**: HTML5 required attribute validation

### Social Login
- Facebook, Google, and LinkedIn integration options
- Icons with accessibility labels

### Responsive Design
- Desktop: Side-by-side forms with sliding panel
- Mobile: Stacked forms with toggle buttons
- Fluid layout that adapts to different screen sizes

### Animations
- Smooth sliding panel transitions
- Hover effects on buttons and links
- Gradient background animations

## Browser Compatibility

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers (iOS Safari, Chrome Mobile)
- Requires JavaScript for full functionality
